# Video Script

## Intro

Joe Speaking is a Gemini-powered live speaking coach built for the Live Agents category. Instead of recording into a static prompt, the learner speaks with Gemini in real time.

## Showcase

The judge story is simple: open the demo, start a live conversation, let Gemini ask the first question, answer with the microphone, get a coaching recap, then retry the same topic and compare the new version to the prior attempt.

## Infrastructure

This challenge build is hosted on Google Cloud Run, uses the Google GenAI SDK, and runs as a public judge demo without any production-data dependency.

## Close

The key product idea is not just live conversation. It is measurable improvement on the same speaking topic, with the full flow visible in one clean judge demo.
